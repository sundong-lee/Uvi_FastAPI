# Hello FastAPI 프로젝트

이 프로젝트는 FastAPI를 사용하여 기본적인 웹 서버를 만드는 예제입니다.

## 설치

먼저, 필요한 패키지를 설치하세요:

```bash
pip install -r requirements.txt
```

## 실행

서버를 실행하려면 다음 명령을 사용하세요:

```bash
uvicorn main:app --reload
```

서버가 시작되면 브라우저에서 `http://127.0.0.1:8000`으로 접속하여 "Hello World!" 메시지를 확인할 수 있습니다.

## API 문서

FastAPI는 자동으로 API 문서를 생성합니다. `http://127.0.0.1:8000/docs`에서 확인할 수 있습니다.# Uvi_FastAPI
